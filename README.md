# TCC - Analise preditiva e monitoramento de custos da dialise no SUS

Este repositorio reune os arquivos iniciais do TCC sobre analise preditiva e monitoramento de custos da dialise no SUS, com foco no desenvolvimento de um dashboard para apoio a gestao em saude.

## Tema

Analise preditiva e monitoramento de custos da dialise no SUS: desenvolvimento de um dashboard para apoio a gestao em saude.

## Conteudo inicial

- `analise.ipynb`: notebook inicial de tratamento e analise dos dados.
- `estrutura tcc - isabelly.docx`: documento com estrutura do TCC.
- `sia_cnv_*.csv`: arquivos exportados do TabNet/DATASUS SIA/SUS.

## Proximos passos

1. Organizar os dados brutos e tratados.
2. Filtrar procedimentos relacionados a dialise.
3. Construir indicadores historicos de custo e quantidade aprovada.
4. Testar modelos preditivos simples.
5. Desenvolver dashboard para visualizacao e apoio a gestao.
